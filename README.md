Welcome to the **Dual Mode Vision-Guided Robotic Arm** project repository! This repository contains the source files, models, and interactive presentation to track the progress of the robotic arm development.

### How to browse this project:
- **Interactive Web Presentation:** The best way to view the project's progress, 3D models (STL viewers), and slides is via the [GitHub Pages Link](https://chepurinatraj.github.io/projectprogress/). Please visit this link first.
- **Repository Structure:**
  - `docs/`: Includes markdown documentation, scripts, and downloadable files like PPTs and PDFs.
  - `models/`: Contains the CAD models (`.step`, `.stl`) of the robotic arm.
  - `scripts/`: Contains Python scripts for server and presentation generation.
  - `Media/`: Images and videos used across the presentations.
  - `index.html` & `slides_data.js`: The frontend code serving the interactive GitHub Pages site.

### Technologies Used:
* Three.js for 3D STL rendering
* HTML/CSS/JS for an interactive progress report interface
* Python scripts for automation

### Local Setup (Optional)
If you want to view the presentation locally including the local server dependencies:
1. Clone this repository: `git clone https://github.com/chepurinatraj/projectprogress.git`
2. Start the local server by running the python script: `python scripts/server.py`
3. Navigate to `http://localhost:8000` (or whichever port your server assigns) in your browser.
